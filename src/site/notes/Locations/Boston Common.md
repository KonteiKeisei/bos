# Boston Common

> [!info] Location Summary
> **District / Neighborhood:** Downtown
> **Type:** Public Park / Gathering Space
> **Era:** 1986
> **Status:** Mundane & Magical

## Description

![[Boston Common.webp|Boston Common|500]]

America's oldest public park, established 1634. Originally used for grazing livestock, it evolved into a hub for public speeches, revolutionary rallies, and civic engagement. Notable speakers include Martin Luther King Jr. The Massachusetts State House overlooks this expansive green space.

---

## History

Established 1634. Site of revolutionary rallies, Vietnam War protests (1960s–70s), and countless civic events.

---

## Key Features

- Expansive green lawns and tree-lined paths
- Frog Pond (skating in winter)
- Civil War memorial
- Views of the Massachusetts State House
